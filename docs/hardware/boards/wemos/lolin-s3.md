# Wemos Lolin S3

!!! success "Fully compatible"

    This product is fully compatible with OpenShock.

- [Official webpage](https://www.wemos.cc/en/latest/s3/s3.html)

## Specifications

- ESP32-S3-WROOM-1
- 16MB Flash (Quad SPI)
- 8MB PSRAM (Octal SPI)
